# mywant-gmail-list-unread-plugin

MyWant custom type plugin for listing unread important Gmail emails.

## Installation

```bash
cd ~/.mywant/custom-types
git clone https://github.com/onelittlenightmusic/mywant-gmail-list-unread-plugin
```

## Usage

```yaml
metadata:
  name: check_gmail
  type: gmail_list_unread
```

## Requirements

- Python 3, Playwright, Chrome with remote debugging (port 9222)
